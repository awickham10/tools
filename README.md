# Tools

A collection of useful single-page web applications hosted on GitHub Pages.

**Live site:** https://awickham10.github.io/tools/

## Available Tools

- **[Auto Loan Calculator](https://awickham10.github.io/tools/auto-loan-calculator/)** - Calculate monthly payments or determine the maximum vehicle price you can afford

## Adding a New Tool

1. Create a new folder with a kebab-case name (e.g., `my-new-tool/`)
2. Add your single-page app as `index.html` inside the folder
3. Add a link to the tool in `index.html` at the root

## Structure

```
tools/
├── index.html              # Landing page
├── auto-loan-calculator/
│   └── index.html
├── [your-new-tool]/
│   └── index.html
└── README.md
```
